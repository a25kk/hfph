Tables

Verwendung der Tabellen

Tabellen finden in folgender Darstellung Verwendung. Von unnötigem Tabelleneinsatz wird abgeraten. Tabellen werden zwar mobil optimiert, sind aber per se nicht mobil tauglich.

Beispiele

Eine Tabelle ohne Rahmen

# Vorname Nachname Benutzername
1 Mark Twain @mtwain
2 Jacob Black @jblack
3 Leo Stein @lstein
{% highlight scss %} ...
{% endhighlight %}

Eine Tabelle mit dünnen Rahmen

# Vorname Nachname Benutzername
1 Mark Twain @mtwain
2 Jacob Black @jblack
3 Leo Stein @lstein
{% highlight scss %} ...
{% endhighlight %}
