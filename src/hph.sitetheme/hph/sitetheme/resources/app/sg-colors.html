Farbschema

Markenspezifische Farbwerte und Definitionen

Standardfarben

Grauwerte

Graustufen werden im Hinblick auf die Barrierefreiheit mit möglichst hohem Kontrast auf Basis eines 100%igen Schwarzwertes automatisch generiert. Da bei der Darstellung auf Monitoren ein reines Schwarz als zu kontrastreich angesehen wird, ist ein dunkler Grauwert $gray-dark als Standard für Fließtext in Verwendung.

{% highlight scss %} $gray-base: #000 !default; $gray-darker: lighten($gray-base, 13.5%) !default; // #222 $gray-dark: lighten($gray-base, 20%) !default; // #333 $gray: lighten($gray-base, 33.5%) !default; // #555 $gray-light: lighten($gray-base, 46.7%) !default; // #777 $gray-lighter: lighten($gray-base, 93.5%) !default; // #eee {% endhighlight %}

Markenfarben

Hauptfarben der Marke werden automatisch als Highlightfarben verwendet und sollten möglichst kontrastreich sein.

Primäre Farbe

Hauptfarbe für Überschriften und den Markennamen.

#274d38
{% highlight scss %} $app-primary-base: rgba(5, 81, 125, 1) !default; $app-primary-darker: darken($app-primary-base, 20%) !default; $app-primary-dark: darken($app-primary-base, 15%) !default; $app-primary: $app-primary-base !default; // #05517D $app-primary-light: lighten($app-primary-base, 15%) !default; $app-primary-lighter: lighten($app-primary-base, 20%) !default; {% endhighlight %}

Sekundäre Farbe

Hauptfarbe für Unterüberschriften und Links

#588c73
{% highlight scss %} $app-secondary-base: rgba(55, 116, 151, 1) !default; $app-secondary-darker: darken($app-secondary-base, 15%) !default; $app-secondary-dark: darken($app-secondary-base, 7.5%) !default; $app-secondary: $app-secondary-base !default; // #377497 $app-secondary-light: lighten($app-secondary-base, 7.5%) !default; $app-secondary-lighter: lighten($app-secondary-base, 15%) !default; {% endhighlight %}

Verwendung der Highlightfarben für Sektionen

Die Highlightfarben sollten in der Navigation der Seite zur besseren Orientierung innerhalb der Teilbereiche dienen. Demnach müssen die Farbwerte einheitlich an identischen Stellen im Portal eingesetzt werden.

Definition der Kontrastwerte hinsichtlich der Barrierefreiheit gemäß WCAG 2.0

sehr gut
Erreicht eine AAA Level Einstufung: bei allen Schriftgrößen ist ein optimaler Kontrast gegeben
gut
Erreicht für Schriftgrößen ab 18pt eine AAA Level Einstufung und bei kleineren Schriftgrößen eine gute AA Bewertung
akzeptabel
Erreicht eine AA Level Einstufung für Schriftgrößen ab 18pt. Bei kleineren Schriftgrößen ist die Lesbarkeit bereits deutlich beeintächtigt bis nicht mehr gegeben.
unzureichend
Erreicht keinen ausreichenden Kontrast, um Menschen mit einer Farbenblindheit, Achromatopsie oder Achromasie eine ausreichende Lesbarkeit gewährleisten zu können.
